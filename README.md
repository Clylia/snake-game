[Online Visit Here](https://snakygame.herokuapp.com/)

## How to local run:

### Dependencies

1.Install Rust and NodeJS

2.Install "wasm-pack"

```bash
$ cargo install wasm-pack
```

### Run

```bash
$ cd snake-game && ./run.sh
```

Open the browser on localhost:3000
