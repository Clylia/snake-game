# snake-game

Dev:

```bash
$ cd snake-game

$ cargo install wasm-pack

$ wasm-pack build --target web

$ cd www

$ npm install

$ npm run dev
```

Prod:

```bash
$ cd snake-game

$ cargo install wasm-pack

$ wasm-pack build --target web

$ cd www

$ npm install

$ npm build

$ cd ..

$ npm start

```
